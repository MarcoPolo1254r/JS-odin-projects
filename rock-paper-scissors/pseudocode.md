Rock-Paper-Scissors Game

--- Flow of the Code ---

User reads the instructions of the game.
User chooses between the proceed button and more info button.
More info button will take the user to another webpage with more instructions for the game.
After proceed User has to input his/her choice (Rock,Paper or Scissors).
Console validates the input before proceeding.
If input is invalid throw an error message "invalid input, please choose between Rock, Paper or Scissor.
After proceed the other oponent(computer) will display a random value.
Compare both values and see who if the user wins of loses.
Dispay a message according to the result
User Can choose to play again if he press the button "play again".
